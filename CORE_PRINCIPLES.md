# Task Leaders — Core Principles

**Date:** March 7, 2026

## The Mission
Build a service marketplace that:
1. Solves a real problem for customers (finding fast, reliable help)
2. Competes with big players (TaskRabbit, Thumbtack) by being leaner
3. Operates as a **one-person operation** — minimal overhead, maximum automation

## Competitive Advantage

| Big Players | Task Leaders |
|-------------|--------------|
| Heavy operations teams | One person + automation |
| 20%+ customer fees | Free for customers |
| Commission-based | Subscription-based |
| Complex vetting | Simple video call |
| Expensive customer acquisition | Organic + word of mouth |

## Operating Principles

### 1. Minimize Manual Work
- WhatsApp handles all communication (no platform messaging to maintain)
- Customer-sourced metrics (no manual review moderation)
- Self-serve provider signup (only founding providers get video calls)
- Automated follow-ups (24-48h after inquiry)

### 2. Minimize Costs
- GitHub Pages = free hosting
- WhatsApp Business API = pay per message (pennies)
- No office, no employees, no inventory
- Stripe for subscriptions (2.9% + 30¢)

### 3. Minimize Complexity
- No native apps (web only)
- No real-time chat (WhatsApp only)
- No dispute resolution (customers/providers handle directly)
- No insurance/guarantees (provider reputation is the guarantee)

### 4. Maximize Automation
- Rankings auto-calculate from metrics
- Provider onboarding → form → auto-approve (after founding period)
- Customer feedback → auto-request via WhatsApp
- Subscription billing → auto-renew, auto-cancel
